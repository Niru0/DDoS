WHAT IS A DDoS ATTACK?

DDoS means "Distributed Denial-of-Service". It floods a server with traffic to make it crash.

HOW IT WORKS?

Thousands of infected devices (botnet) flood a server with traffic, making it crash.

TYPES OF DDoS ATTACK

  1. UDP Flood
  2. SYN Flood
  3. HTTP Flood
